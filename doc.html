Классы

InputEngine

Keys() - в этой функции есть только один аргумент. Он принемает любую клавишу из списка Keys (Этот список является отдельным классом). Возврощает либо false (если не нажата), либо true (если клавиша нажата).

MousePos() - возврощает позтцтю мыши в моссиве

MouseClick() - возврощает массив (нажата лкм и нажата пкм), выглядит так [лкм, пкм]

SetMousePos() - задаёт позицию мыши (имеет 2 аргумента x и y, туда надо передать Number)

SetMouseVisable() - задаёт видемость мыши (имеет одн аргумент в который нужно передать true или false)
