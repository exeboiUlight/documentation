ULight Engine API Documentation

InputEngine

Класс для работы с вводом (клавиатура и мышь)

Keys(key)
Параметры: key (string) - клавиша из списка Keys
Возвращает: boolean - состояние клавиши (нажата/не нажата)
Проверяет состояние указанной клавиши. Возвращает true, если клавиша нажата, и false в противном случае. 
if (InputEngine.Keys(Keys.space)) {
    // Действие при нажатии пробела
}
MousePos()
Возвращает: [number, number] - позиция мыши [x, y]
Возвращает текущую позицию курсора мыши в виде массива [x, y].
MouseClick()
Возвращает: [boolean, boolean] - состояния кнопок [ЛКМ, ПКМ]
Возвращает состояния кнопок мыши: первый элемент массива - левая кнопка, второй - правая.
SetMousePos(x, y)
Параметры: x (number), y (number) - новые координаты
Устанавливает позицию курсора мыши в указанные координаты.
SetMouseVisable(visible)
Параметры: visible (boolean) - видимость курсора
Управляет видимостью курсора мыши. Принимает true для показа и false для скрытия.

Engine

Основные функции движка

setOS(os)
Параметры: os (string) - "Win" или "Web"
Устанавливает целевую операционную систему для разработки игры.
WindowSize()
Возвращает: [number, number] - размер окна [ширина, высота]
Возвращает текущий размер игрового окна.
Quit()
Завершает работу приложения.
System(text)
Параметры: text (string) - системный текст
Устанавливает системный текст (назначение зависит от реализации).

Camera

Управление камерой

SetCameraPos([x, y, z])
Параметры: [x, y, z] (number[]) - новая позиция
Устанавливает позицию камеры в 3D пространстве.
SetCameraRotate([x, y, z])
Параметры: [x, y, z] (number[]) - новые углы вращения
Устанавливает вращение камеры по осям X, Y, Z.
GetCameraPos()
Возвращает: [number, number, number] - позиция камеры [x, y, z]
Возвращает текущую позицию камеры.
GetCameraRotate()
Возвращает: [number, number, number] - вращение камеры [x, y, z]
Возвращает текущее вращение камеры.

Time

Работа со временем

Time()
Возвращает: number - текущее время
Возвращает текущее время игры (в секундах).
DeltaTime()
Возвращает: number - время между кадрами
Возвращает время, прошедшее между последним и текущим кадром (в секундах). Полезно для плавной анимации.
FPS()
Возвращает: number - текущий FPS
Возвращает текущее количество кадров в секунду.

VariableSystem

Система работы с игровыми объектами

RenderObjects()
Очищает списки объектов и текстов для рендеринга.
NewSprite(textureEnumerate, position, scale, rotate)
Параметры: textureEnumerate (number), position ([x, y, z]), scale ([x, y, z]), rotate ([x, y, z])
Создает новый спрайт для рендеринга.
NewText(mytext, position, scale, color, idfont)
Параметры: mytext (string), position ([x, y, z]), scale (number), color (string), idfont (number)
Создает новый текст для рендеринга.
NewMesh(vertices, indices, normals, tex_coords)
Параметры: vertices (number[]), indices (number[]), normals (number[]), tex_coords (number[])
Создает новую полигональную модель для рендеринга.

Component

Компонентная система для игровых объектов

ClearScene()
Очищает текущую сцену от всех объектов.
SetScene(scenename)
Параметры: scenename (string)
Возвращает: boolean - успех операции
Устанавливает текущую активную сцену.
GetScene()
Возвращает: string - имя текущей сцены
Возвращает имя текущей активной сцены.
CreateGameObject(name)
Параметры: name (string)
Возвращает: string - ID созданного объекта
Создает новый игровой объект с указанным именем и возвращает его идентификатор.
